# Tests

This directory contains the test suite for the Silent King project.

## Structure

The structure mirrors the `src` directory. Tests for `src/silent_king/foo.clj` should be located in `test/silent_king/foo_test.clj`.

## Running Tests

You can run the tests using the scripts provided in the `scripts/` directory:

- **Windows**: `.\scripts\run-tests.ps1`
- **Linux/macOS**: `./scripts/run-tests.sh`
